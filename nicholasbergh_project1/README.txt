Compiling: use make to compile the program if not using linux (you'll need to type in your OS from the prompt), you'll need clang++ installed

Running: run like so: ./imagequilt [in file] [out file] [square patch size] [# horizontal patches] [# vertical patches] [overlap size] [# source samples per patch]
Example: 			  ./imagequilt input_marble_texture.jpg output_marble_texture.png 70 7 8 15 2000
